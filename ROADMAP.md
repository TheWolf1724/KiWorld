# 🗺️ Roadmap — KiWorld

Plan de desarrollo. No es un compromiso cerrado; las prioridades pueden cambiar.

## ✅ Hecho

- [x] Movimiento top-down 8 direcciones + cámara.
- [x] Mundo con TileMapLayer y colisiones ajustadas.
- [x] Enemigos con IA (aggro / persecución / ataque) y barra de vida.
- [x] Combate: melee, bola de ki (E) y ataque en área (Q).
- [x] Barra de ki con regeneración y carga manteniendo R.
- [x] XP, niveles y elección de 1 de 3 mejoras por nivel.
- [x] Monedas especiales + tienda + skins de personaje.
- [x] Ventana de controles, menú de pausa (Continuar / Reiniciar / Salir).
- [x] Guardado local.
- [x] **Jefe (boss)** con barra de vida grande, dos fases y varios patrones de ataque.

## 🔜 Próximo

- [ ] **Inventario y equipo** + loot al derrotar enemigos.
- [ ] **Más enemigos** y variantes por bioma.
- [ ] **Habilidades nuevas** (dash, carga de energía, transformaciones).
- [ ] **Segunda zona / dungeon** con transición de escenas.

## 🧭 Medio plazo

- [ ] **Audio**: música y efectos de sonido.
- [ ] **NPCs y misiones** sencillas con diálogo.
- [ ] **Control táctil** y export pulido a **Android**.
- [ ] Mejoras de tienda (más skins, vista previa, categorías).
- [ ] Sistema de guardado por slots.

## 🌐 Largo plazo

- [ ] **Multiplayer** (la arquitectura ya está desacoplada de la red).
- [ ] Eventos de mundo / jefes globales.
- [ ] Editor de mapas / contenido.

## 💡 Ideas sueltas

- Rarezas de monedas y cofres.
- Mascotas / compañeros.
- Logros.
